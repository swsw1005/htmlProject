display 속성 (none,inline,block,inline-block)

1. block : 크기와 관계없이 1라인을 차지

안녕하세요 ( div태그 => 1라인 전체 차지 )

2. inline : 자기자신의 크기만큼 자리를 차지

하하하 호호호 ( b span img u 태그 => 자신의 크기만 차지 )

3. block 속성태그 : h1~h6 div ul li hr table 태그등등(네이버참조)

4. block속성를 inline속성으로 (div는 block 태그)
ex) <div> 하하하 </div> <div> 호호호 </div> <div> 헤헤헤 </div>

display 속성추가 전 display:inline 변경후
X display:inline;

5. inline속성을 block으로 (span는 inline 태그)
ex) <span> 하하하 </span> <span> 호호호 </span> <span> 헤헤헤 </span>

display 속성추가 전 display:block 변경후
X display:block;
하하하 호호호 헤헤헤 하하하 호호호 헤헤헤

6. inline-block (inline이 되면서 block의 속성을 가짐)
ex) <div> 하하하 </div> <div> 호호호 </div> <div> 헤헤헤 </div>

display 속성추가 전 display:inline 변경후 display:inline-block 변경후
width:150px;
height:50px;
background:yellow;
border:1px solid blue; 		display:inline;
width:150px;
height:50px;
background:yellow;
border:1px solid blue; 		display:inline-block;
width:150px;
height:50px;
background:yellow;
border:1px solid blue;
