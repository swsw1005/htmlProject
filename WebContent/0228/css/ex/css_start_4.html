CSS의 사용법

문서내의 <style> ~ </style> 내에 사용

<style> h1 {color:blue;font-size:20px} </style> <h1> 오늘은 좋은날 </h1>

오늘은 좋은날

태그내에 style="속성:값;속성;값" 사용

<div style="color:#12ff55;font-size:20px"> 내일도 좋은날 </div>
내일도 좋은날

외부문서를 등록하여 사용

test.css파일 내용
.tt { color:darkblue; font-size:30px; text-decoration:underline; }
<link rel="stylesheet" href="test.css"> <div class=tt> 모레도 좋은날 </div>
모레도 좋은날
