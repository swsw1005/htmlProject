visibility 속성 (visible, hidden, collapse)

1. visible (내용을 보이게 한다.) => 생략과 동일
ex) <div> 안녕하세요 </div>

visibility 속성추가 전 visibility:visible 변경후
X visibility:visible;
안녕하세요
안녕하세요

2. hidden (내용을 숨긴다.)
ex) <div> 안녕하세요 </div>

visibility 속성추가 전 visibility:hidden 변경후
X visibility:hidden;
안녕하세요
안녕하세요

3. collapse (테이블을 숨긴다.)
ex)
<table>
<tr>
<td> 안녕하세요
</tr>
<tr>
<td> 안녕하세요
</tr>
</table>

collapse 속성추가 전 tr태그에 visibility:hidden 변경후 tr태그에 visibility:collapse 변경후
X visibility:hidden; visibility:collapse;
안녕하세요
