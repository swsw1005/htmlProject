CSS의 구조

선택자 선언부
h1 { color: blue; font-size: 20px; }
  속성 값 속성 값

태그이름을 선택자로 사용

<style> h1 {color:blue;font-size:20px} </style> <h1> 오늘은 좋은날 </h1>

오늘은 좋은날

id를 선택자로 사용

<style> #pkc {color:blue;font-size:20px} </style> <div id=pkc> 내일도 좋은날 </div>
내일도 좋은날

class를 선택자로 사용

<style> .pkc {color:blue;font-size:20px} </style> <div class=pkc> 모레도 좋은날 </div>
모레도 좋은날

복수 선택자로 사용

<style> .aa , #bb {color:blue;background:yellow} </style> <div class=aa> 글피도 좋은날 </div> <div id=bb> 글피도 좋은날 </div>
글피도 좋은날
글피도 좋은날
